### Introduction
- Laravel Cheat Sheet 是一款sublime插件
- 适用于使用Laravel框架的PHP开发者
- 它是我收集以及自己整理出来的laravel开发手册（多参考[laravel小抄](http://cheats.jesse-obrien.ca/)）
- 支持laravel最新5.1版本
